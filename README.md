# Library Management System

This project is a simple web-based Library Management System built using HTML, CSS, and JavaScript.  
It allows you to manage books, users, and transactions (issue and return of books) with an interactive dashboard.

## Features

- Add, edit, and delete books
- Register, edit, and delete users
- Issue and return books with transaction tracking
- Dashboard showing statistics:
  - Total books
  - Available books
  - Total members
  - Books issued
- Search functionality for books and users
- Responsive design for mobile and desktop
- Modal-based editing and alert system

## File Structure

- `index.html` – Main HTML file containing the structure of the application
- `style.css` – Stylesheet for layout, design, and responsiveness
- `script.js` – JavaScript logic for handling data, events, and UI updates
